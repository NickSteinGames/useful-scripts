## Installation
### From Assets Lib
  Just open `AssetLib` and print in search:
  ```
  Useful Scripts
  ```
### From wtere
Open [`Releases`](https://github.com/NickSteinGames/useful-scripts/releases) and download [Latest Release](https://github.com/NickSteinGames/useful-scripts/releases/latest).
